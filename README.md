# Arjun Singh Rajput - Personal Portfolio 🚀

A stunning, modern personal portfolio website for **Arjun Singh Rajput**, a multidisciplinary technologist and AI innovator.

## 🌟 Features

- **Modern Design**: Light mode with pastel gradients, rounded corners, and smooth animations
- **Fully Responsive**: Mobile-first approach (375px → 768px → 1024px → 1440px+)
- **Dark Mode**: Smooth transition between light and dark themes
- **Smooth Animations**: Powered by Framer Motion for professional microinteractions
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Accessibility**: WCAG compliant design
- **Performance**: Fast loading times with optimized assets

## 🎨 Sections

1. **Hero Section**: Full-screen introduction with animated gradient text and CTAs
2. **About Me**: Personal bio with profile photo and fun facts
3. **Skills & Capabilities**: Interactive skill cards with hover effects
4. **Projects Showcase**: 7 featured projects including AGENTIX (National Hackathon Winner)
5. **Experience Timeline**: Vertical timeline with work experience and education
6. **Achievements**: Major accomplishments with animated counter (500+ LeetCode)
7. **Contact**: Form with validation and social media links

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (React 18)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Font**: Inter (Google Fonts)

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository or navigate to the project folder:
```bash
cd "c:\Users\Asus\Desktop\Personal Portfolio"
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
